--BUSCAMINAS--
El proyecto se trata de un buscaminas en el que puede seleccionar un tablero de 3x3 a 10x10 en el que puede barrer casillas si presiona la 
opción de Barrer Casilla y presionas una casilla. El poner y quitar banderas debe de desmarcar el botón de barrer casillas y 
presiona el boton de poner/quitar bandera. 
Para ver los commits realizados se situan en las ramas r, LeoBranch y miguel-branch.
